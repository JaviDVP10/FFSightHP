# FFSightHP


Plugin con una base creada para contar el DPS por segundo realizado por tu personaje en final fantasy XIV.

La idea del plugin es que sirva como una base con el DPS del daño del personaje, es un ejemplo sencillo que saca todo el DPS en un solo número. Por ahora solo funciona
para el daño realizado por el personaje, pero se puede expandir para incluir sanación, mitigación, etc.

El idioma del plugin es inglés y un punto a realizar es hacer que sea multilenguaje.
